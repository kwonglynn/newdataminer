Configuration of slug for ease of reference.

1. In views.py of the application.
class ModuleDetailView(DetailView):
    model = Module
    context_object_name = 'module'
    slug_field = 'title'      # 'title' is a field of the model.
    slug_url_kwarg = 'title'
2. In urls.py of the application.
urlpatterns = [
    path('modules/<slug:title>', ModuleDetailView.as_view(), name='module_detail'),
]
3. In template file refering to the detailed view, normally this is in the ListView template.
...
<td><a href="{{ module.title }}">{{ module.title }}</a></td>
...

Remote access to the computer at home with dynamics DNS (DDNS).

1. Apply for a free hostname.

Go to www.noip.com, and apply for an free domain

Specify a user name, the user name will be used later for the router settings.

2. Configure the router for remote access.

Log in to the admin page of the router via http://192.168.10.1

Enable the DynDNS service of the router, and fill in the information for no-ip.com.

Set up Port forwarding on the router:

Home can be any name you like.

Leave the Protocal as TCP

WAN is the port from out side, you can specify any port for this, normally it is good to specify a high port so that it is not easy to be cracked.

LAN is the local port to be mapped with WAN, it should map the port of the actual server on the local computer. For SSH the LAN is 22.

The Destination IP is the local IP what should be mapped.

You can also specify Source IP of the remote computer for security concern.

3. Now it is time to log in from outside!
[x_guaku@tetralith2 ~]$ ssh -p 1022 guanglin@guanglin.ddns.net

Cheers!
